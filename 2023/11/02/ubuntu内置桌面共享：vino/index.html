    使用ubuntu内置桌面共享：vino，实现远程桌面

    sigmapoet 教程库 > linux ubuntu vino vnc
    Created At : 2023-11-02 10:00
    Views 👀 :
    1. 使用场景
    2. 配置步骤
      1. 1、配置vino
      2. 2、配置vnc viewer

    使用场景

    将ubuntu系统的桌面共享给其它用户，通过vnc viewer 5900端口，实现对ubuntu桌面的控制和访问，类似云桌面。

    接下来以我自己的系统为例介绍：ubuntu系统版本：18.04，windows系统：win11，两者通过校园网大局域网互联。

    配置步骤

    1、配置vino

    在ubuntu系统上，输入以下命令，安装配置vino。

    sudo apt-get install vino

    打开ubuntu系统设置->共享->屏幕共享，按下图进行设置。密码最多可以设置8位，网络根据自己实际情况选择。

    然后关闭共享加密策略，

    sudo apt-get install dconf-editor
dconf write /org/gnome/desktop/remote-access/require-encryption false

    或者通过以下命令关闭加密策略，

    gsettings set org.gnome.Vino require-encryption false

    2、配置vnc viewer

    在windows上下载vnc-viewer软件，下载链接：点击链接到下载页面。下载安装完成后，打开vnc-viewer。

    在弹出的页面新建一个连接，在以下页面输入ubuntu系统信息。

    在VNC Server一栏输入ubuntu系统的ip地址和端口，vino默认使用端口5900，例如：192.168.1.77:5900。

    在Name一栏输入名称，例如myUbuntuServer。输入完成后点击OK。

    点击生成的共享桌面，在弹出的页面输入ubuntu系统登陆密码。输出完成后点击OK，就会弹出远程分享的ubuntu桌面了。
